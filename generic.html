July 8, 2023

Mom and Pops Pizza Shop

This was a team project where we developed a website where customers could order pizza, drinks, and sides. The orders were sent to the shop and delivery was not available with this design. We used the Django Framework and used Python as our primary language. This project also included using HTML/CSS/JavaScript. This was my first ever team proejct I built in school. It was also the first time I had the opportunity to practice using an Agile Methodology. We used a backlog for all of our tasks that were needing to completed. We'd then meet each week and divide one task up between each of us (there were 4 of us working on this project). My main job on this project was building the payment page, where I used Sandbox in order to allow for fake payments to be accepted.
